# Assets Directory

This directory should contain the following image assets for the app:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon

For now, these are optional and Expo will use default assets. You can add your custom assets here when ready.
